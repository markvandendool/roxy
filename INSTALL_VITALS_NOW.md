### Method 1: Browser (EASIEST - RECOMMENDED)
1. Browser should open automatically to: https://extensions.gnome.org/extension/1460/vitals/
2. If not, visit that URL manually
3. Click the toggle switch to "ON" (it will install automatically)
4. If prompted, install "GNOME Shell Integration" browser extension
5. Restart GNOME Shell: Press `Alt+F2`, type `r`, press `Enter`
